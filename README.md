# Este repositório contém todos os arquivos Python utilizados durante o desenvolvimento do projeto

A estrutura de pastas pode ser alterada; entretanto, certifique-se de que os caminhos utilizados em cada um dos arquivos Python correspondam à pasta ou ao arquivo de destino.

## Aviso importante

O arquivo `merge_tox_emo.py` só deve ser utilizado caso os IDs presentes no seu arquivo `.csv` estejam consistentes. Não o utilize em caso de IDs repetidos.

### Recomendações

É recomendado o uso da ferramenta **pyenv**, pois ela facilita a troca entre versões do Python. Caso utilize **Anaconda**, certifique-se de que as versões das bibliotecas utilizadas nos arquivos Python estejam atualizadas e compatíveis.

No diretório `sentimento-liwc` encontram-se alguns testes preliminares com dicionários, utilizando as bibliotecas **nrclex** e **spaCy**, devido ao fato de o software **LIWC-22** exigir licença.
